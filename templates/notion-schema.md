# Notion 데이터베이스 스키마

## 붙여넣기용 표 형식

| 날짜 | 카테고리 | 트렌드 키워드 | 출처 | URL | 조회수/반응 | 핵심 요약 | 브런치 글감 | 점수 | 상태 | 후속 액션 |
|---|---|---|---|---|---|---|---|---|---|---|
| YYYY-MM-DD |  |  |  |  |  |  |  |  | 수집 |  |

## 카테고리 값

- AI 실무
- 1인 창업/부업
- 정부지원사업
- 콘텐츠 수익화
- 해외 트렌드
- YouTube
- 뉴스

## 상태 값

- 수집
- 분석 완료
- 글감 선정
- 초안 작성
- 발행 완료
- 보류

## Notion API 연결 시 주의사항

- 데이터베이스에 새 페이지 생성 전 중복 URL을 조회한다.
- 동일 URL이 있으면 새로 만들지 않고 기존 항목을 업데이트한다.
- 생성일, 수정일, rollup 등 Notion 자동 생성 속성은 직접 쓰지 않는다.
- API 실패 시 Markdown 표를 대체 산출물로 생성한다.
